# TradingView Pine Scripts

این مخزن شامل اسکریپت «Fixed Range Volume Profile» بدون محدودیت تعداد کندل است.

## Unlimited Fixed Range Volume Profile
- فایل: `scripts/unlimited_fixed_range_volume_profile.pine`
- بر پایه Pine Script نسخه ۵
- امکان توزیع حجم روی محدوده دلخواه بدون سقف تعداد کندل‌ها
- دارای تنظیمات برای تعریف شروع و پایان محدوده، گام قیمتی (با اجباری بودن حداقل تیک)، سقف تعداد سطوح قیمتی، بیشینه عرض هیستوگرام و نمایش برچسب‌های POC/VAH/VAL
- محدودیت پیش‌فرض آبجکت‌ها در اندیکاتور بالا برده شده تا هیستوگرام‌های بزرگ بدون خطای «Too many boxes» اجرا شوند

برای استفاده، فایل را در بخش Pine Editor تریدینگ‌ویو کپی کنید و تنظیمات دلخواه را اعمال نمایید.
